### 2차원 배열
- 1차원 배열의 연장선
- **x축과 y축** 같은 형태로 많은 문제에서 2차원 배열은 사각형의 형태로 다룰 수 있다.
- **원소 접근** 2차원 배열에 y 좌표로 먼저 접근한 후 x 좌표로 접근해야 올바르게 원소에 접근이 가능하다.
