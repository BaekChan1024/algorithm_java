# 삼각 달팽이

## 상수조건
- 밑변의 길이와 높이가 n인 삼각형에서 맨 위 꼭짓점부터 반시계 방향으로 달팽이 채우기를 진

## 풀이
- 아래 방향, 오른쪽 방향, 위쪽 방향으로 이동하면서 채워나감
- 이동할 때마다 이동할 수 있는 만큼 이동하고, 이동할 수 없는 경우 방향을 바꿈

## 풀이실수

- 1st
    - 아래 방향, 오른쪽 방향, 위쪽 방향을 Loop 형태로 구현하려다 보니 고려해야하는 케이스가 많아짐
    - 방향을 파악하고 구현해야 할듯
    - 답봄

- 공간 복잡도: O(N^2)
